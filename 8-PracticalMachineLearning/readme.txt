This is a repository created for Coursera - Practical Machine Learning - Prediction Assignment.

PML-PA-1.Rmd is R markdown file written in RStudio. It can be viewed by clicking on its name.

PML-PA-1.html is html file knited from RStudio. It can be viewed by any of the following methods.

Method 1:

- Click on its name.
- This will display its source code.
- Click on the "Open this file in GitHub for Windows" icon next to the History button in the box above its source code.
- This will download and display the file.

Method 2:

- Write "http://htmlpreview.github.io/?https://github.com/Lapyl/PracticalMachineLearning/blob/master/PML-PA-1.html" in the URL box of your Internet browser.

Method 3:

- Write "http://htmlpreview.github.io" in the URL box of your Internet browser.
- Write "https://github.com/Lapyl/PracticalMachineLearning/blob/master/PML-PA-1.html" in the dialog box of the resulting web page.
